## Highlights

- **Full-lifecycle ownership** — UI/UX → Frontend → Backend → AI → deployment.
- **The training solution** framed as a closed loop: Create → Learn → Analytics.
- **Selected work with real metrics** — Trainizi (15K+ learners, −30% dev time, #1 AI solution in ASEAN), Wolffun AI companion, nano/micro-learning apps, a real-time F&B POS, and full-stack e-commerce platforms.
- **Single hand-coded `index.html`** — no framework, no build step, zero dependencies.

## Built with

- Plain **HTML + CSS + vanilla JavaScript**, contained in one file.
- Type system: **Montserrat · Inter · JetBrains Mono** (Google Fonts).
- Tech-stack logos via **Devicon** (CDN).
- **Bilingual VI/EN** toggle, persisted in `localStorage`.
- Responsive layout, **WCAG 2.1 AA** color contrast, reduced-motion support.

## Run locally

No install and no build — just open the file:

```bash
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```

## Deploy (GitHub Pages)

1. **Settings → Pages**
2. **Source:** Deploy from a branch → `main` / `/ (root)` → **Save**
3. The site goes live at `https://<username>.github.io/portfolio/`
